# 🧪 Digital Audio Lab 
オーディオ信号処理とFFT（高速フーリエ変換）アルゴリズムを多言語で探求するリポジトリです。

## 🎯 プロジェクトの目的
-オーディオデータの「帯域幅」と「解像度」の関係を深く理解する。
- リアルタイム性とパフォーマンスの観点から、プログラミング言語（JS, Python, C++）による実装の違いを分析する。

## 📂 ディレクトリ構成
- **[JavaScript](./JavaScript)**: `Web Audio API` と `Canvas` を用いたブラウザベースのリアルタイム・ビジュアライザー
- **[Python](./Python)**: `NumPy`, `SciPy` を活用したデータ分析およびFFTアルゴリズムのプロトタイピング
- **[Cpp](./Cpp)**: `JUCE` フレームワークや標準ライブラリを用いた高性能・低遅延のリアルタイム処理

## 🛠 技術スタック / Tech Stack
- **Languages**: JavaScript, Python, C++

  - FFT (Fast Fourier Transform)
